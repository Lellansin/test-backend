Load Middlewares

Load Controllers

After the global middlewares loaded, it will load all the controllers which defined in the controller's index.

Host

While all the middlewares and controllers loaded, env NODE_ENV will be check if it's 'production'.

If we started from 'production', app will host on the port which specified in configuration file (default located src/config/config.default.ts).

Otherwise, just export app for test.
